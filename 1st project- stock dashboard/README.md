# Quant Finance Project 1 – Stock Analytics Dashboard

**Goal:**  
Analyze the risk–return performance of three US stocks using historical price data.

**What it does:**  
- Downloads price data from Yahoo Finance  
- Calculates annualized return, volatility, and Sharpe ratio  
- Plots cumulative performance since 2022  

**Skills demonstrated:**  
Python (pandas, numpy, matplotlib, yfinance), financial statistics, data cleaning & visualization.

**Findings:**  
From 2022–2025, NVIDIA displayed the highest returns but also the greatest volatility, while Microsoft provided steadier risk-adjusted performance.
